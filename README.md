### [平台描述](https://github.com/OS-Q/D99/wiki) 

无线管道通信平台，作为无线射频数据调制设备，完成数据的管道通信

所有的数据管道都需要通过一个系统封装成标准的通信数据协议

输入方式兼容，包括兼容的UART/SPI/IIC三种，具有协议分流和命令统一的优势
